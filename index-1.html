SNAKE RELOADED

Dev: Oscar Lopez

Snake Reloaded is an app made from

data structures on the language Python.

Source Code: https://github.com/olgt/EDD_1S2019_P1_201612341

This app was made with specific data structures using the language python. It was tested on Windows 10. on August, 2019.

This application is still on alpha.

This app is intended to be a game which other programmers can play and also see the source code. So that they can actually have this as a guide when programming data structures.

Info: '+' This is a plus point, it will add a node to your snake '*' This is a less point, it will remove the last node on your snake

Controls: Up Key: Moves the snake upwards Down Key: Moves the snake down Left Key: Changes direction to the left Right Key: Changes direction to the right
